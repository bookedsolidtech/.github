### 🧪 HELiXiR — AI-Powered Component Intelligence

> Give AI agents full situational awareness of any web component library.

[![npm version](https://img.shields.io/npm/v/helixir?logo=npm)](https://www.npmjs.com/package/helixir)
[![npm downloads](https://img.shields.io/npm/dw/helixir)](https://www.npmjs.com/package/helixir)
[![Build](https://img.shields.io/github/actions/workflow/status/bookedsolidtech/helixir/build.yml?branch=main&label=build&logo=github)](https://github.com/bookedsolidtech/helixir/actions)
[![Tests](https://img.shields.io/github/actions/workflow/status/bookedsolidtech/helixir/test.yml?branch=main&label=tests&logo=github)](https://github.com/bookedsolidtech/helixir/actions)

HELiXiR is an MCP server that reads your Custom Elements Manifest and design tokens, giving AI coding assistants **accurate** component APIs instead of hallucinated ones. No more invented attributes. No more guessed slot names.

- Works with **any** web component library (Shoelace, Spectrum, FAST, HELiX, your own)
- Reads Custom Elements Manifest (CEM) for properties, events, slots, CSS parts
- Ingests W3C DTCG design tokens
- MCP protocol — plug into Claude, Cursor, Windsurf, or any MCP-compatible tool

```bash
npx helixir init     # Generate starter config
npx helixir          # Start the MCP server
```
